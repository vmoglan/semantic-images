wallpaper

Image processing

This application offers a few image processing tools such as resize and crop in current v0.0.1. More tools, including filters, are to come in a future version.

timeline

Semantic annotation

Although not available in v0.0.1, the next step in development is to provide a series of features that allow the semantic annotation of an image or a collection of images for usage with SPARQL queries.



Background

This project is developed by Vlad Moglan within the context of the interface programming course at the Hochschule Mainz.
